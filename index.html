Prework Study Guide

HTML

  • The HTML acronym stands for Hypertext Markup Language
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Stanford University HTML Cheat Sheet
  • We can use semantic HTML elements to indicate the role of their content, as well as to give structure to the content of the webpage.
  • link elements are used to create a connection to a external file

CSS

  • The CSS acronym stands for Cascading Style Sheets
  • CSS is a programming language used by developers to define how webpage content is presented to users.
  • There are three ways to style a webpage using CSS: inline CSS, internal CSS style sheet, and external CSS style sheet.
  • When a developer wants to style an element directly in the HTML file, they use inline CSS.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • ".card" class selector was added in "style.css" doc to designate class that will be used in html doc "index.html"

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git checkout: will allow you to switch to another branch and check it out into your working directory.
  • git pull: will fetch and merge any commits from the tracking remote branch
  • git pull origin main: will pull down the latest main branch.
  • Complete Git Flow: it's important to frequently sync the local version of your code with the remote version in GitHub.
